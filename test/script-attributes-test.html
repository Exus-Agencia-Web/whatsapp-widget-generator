Prueba de Widget WhatsApp con Atributos en Script

Inicialización con atributos en la etiqueta script
El widget debe inicializarse automáticamente usando los atributos definidos en la etiqueta script. 
<script
  async 
  src="../whatsapp-widget/widget.js"
  phone="519900012345"
  brand="LiveConnect Test"
  color="#FF5722"
  bubble-text="Prueba de atributos"
  welcome="¡Hola! Esta es una prueba de inicialización con atributos en script."
  position="bottom-right"
  delay="3"
></script>
